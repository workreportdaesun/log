설정

설정 값은 자동으로 저장되어 다음에 열 때도 유지됩니다. work-shoot/work-gallery와 동일한 구분을 사용합니다.

갤러리(Supabase)에서 불러오기

work-gallery에 업로드된 사진을 목록에서 선택하면, 기기번호/작업구역/작업내용/날짜가 자동으로 채워진 상태로 페이지가 생성됩니다. 위 설정에서 "사진관리 구분"을 선택해두면 갤러리 불러오기가 해당 구분으로 기본 필터된 채로 열립니다.

"PC에서 선택 불러오기"는 갤러리 불러오기처럼 선택한 사진들을 위에서 고른 레이아웃(2장/4장)대로 묶어 페이지를 만듭니다. "개별 불러오기"는 선택한 사진 각각을 별도 페이지로 만들어, 서로 관련 없는 사진을 한 장씩 넣을 때 편리합니다.

개별 삽입

개별 삽입 (페이지 목록)

위 갤러리 불러오기로 만들어진 페이지를 여기서 개별 수정하거나, 페이지를 직접 추가해 사진을 한 장씩 따로 넣을 수 있습니다. 파일명이나 사진의 EXIF 설명 정보로 의미 있는 내용을 추정할 수 있으면 "작업내용" 필드에 자동으로 채워집니다.
